# UExplorer - UCoreFS File Manager

A Directory Opus-inspired file manager built with PySide6, showcasing the Foundation template and UCoreFS filesystem database.

## Features

- **Dual-Pane Browser** - Independent left/right file panes
- **Tag Management** - Hierarchical tags with synonyms
- **Smart Albums** - Dynamic query-based collections
- **Detection Viewer** - Bounding box overlays
- **Rules Engine** - Automated file organization
- **Visual Query Builder** - No-code search interface

## Requirements

```bash
pip install PySide6 qasync
```

## Running

```bash
cd samples/uexplorer
python main.py
```
