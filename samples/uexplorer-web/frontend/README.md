# UExplorer Web Frontend

Svelte-based frontend for the UExplorer Web file manager.

## Features

- Modern, responsive UI
- Directory browsing with custom components
- Real-time search
- File and directory navigation
- Custom path navigation

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Frontend will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Components

- **DirectoryBrowser**: Main file/directory browsing interface
- **SearchBar**: File search with live results
