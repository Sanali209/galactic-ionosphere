# UExplorer Overview

UExplorer is the flagship application for the USCore framework. It is a "smart file manager" designed for creative professionals who manage large asset libraries.

## Key Features

| Feature | Description | Doc Link |
| :--- | :--- | :--- |
| **File Browser** | High-performance card view with thumbnails. | [File Browser](features/file_browser.md) |
| **Search** | Semantic and metadata search. | [Search Panel](features/search_panel.md) |
| **Metadata** | Edit tags, ratings, and descriptions. | [Metadata Editor](features/metadata_editor.md) |
| **Rules** | Automated organization rules. | [Rules Engine](features/rules_engine.md) |

## Application Structure

UExplorer follows a **Dual-Pane** philosophy (inspired by Directory Opus) but modernized with a docking interface.

-   **Center**: File Browser tabs.
-   **Left**: Navigation (Directories, Tags, Albums).
-   **Right**: Inspector (Metadata, Relations).
-   **Bottom**: Dashboard / Status.
